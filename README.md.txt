# Netflix Data Analysis using SQL

## 📌 Project Overview

This project analyzes the Netflix dataset using PostgreSQL. It demonstrates SQL skills including data exploration, data cleaning, business analysis, subqueries, Common Table Expressions (CTEs), and window functions.

## 🛠️ Tools Used

- PostgreSQL
- pgAdmin 4
- SQL

## 📂 Dataset

- Netflix Titles Dataset (CSV)

## 📁 Project Structure


## 📊 Business Questions Solved

- Count Movies and TV Shows
- Find the most common rating
- Find the oldest movies
- Find content added each year
- Find the longest and shortest movies
- Find movies longer than the average duration
- Analyze movie duration
- Categorize content using CASE statements
- Use CTEs for reusable queries
- Apply Window Functions (ROW_NUMBER)

## 🧹 Data Cleaning Performed

- Checked missing values
- Checked duplicate records
- Checked blank values
- Converted text data into proper format
- Used CAST() and REPLACE() for duration analysis

## 💡 SQL Concepts Used

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- HAVING
- CASE WHEN
- Aggregate Functions (COUNT, MIN, MAX, AVG)
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions (ROW_NUMBER)
- CAST()
- REPLACE()
- TRIM()
